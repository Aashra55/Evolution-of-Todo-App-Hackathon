## Overview

This project is an AI-powered conversational interface designed to manage user todos through natural language interaction. This repository contains the monorepo structure for the frontend, backend, and AI agent components developed as part of Phase III (Basic Level) of the Evolution of Todo Hackathon.

## Components

*   **`backend/`**: Python FastAPI application for API endpoints, MCP server, and data persistence.
*   **`frontend/`**: Node.js application (using OpenAI ChatKit) for the conversational UI.
*   **`ai-agent/`**: Python application (using OpenAI Agents SDK) for natural language processing and tool orchestration.
